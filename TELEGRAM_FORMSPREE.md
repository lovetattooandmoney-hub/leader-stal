# Заявки в Telegram через Formspree (бесплатно)

Без Cloudflare, Node.js и программирования.

## Шаг 1. Регистрация

1. Откройте [formspree.io](https://formspree.io)
2. Нажмите **Get Started** и зарегистрируйтесь

## Шаг 2. Создание формы

1. В кабинете нажмите **New Form**
2. Название: например, «КБ Лидер‑Сталь — заявки»
3. Нажмите **Create Form**

## Шаг 3. Привязка Telegram

1. Откройте созданную форму
2. Вкладка **Plugins** (или **Integrations**)
3. Найдите **Telegram** → **Connect**
4. Подтвердите вход в Telegram
5. Formspree отправит тестовое сообщение — проверьте Telegram

## Шаг 4. Копирование URL формы

В настройках формы найдите URL вида:
```
https://formspree.io/f/xxxxxxxx
```

Скопируйте его полностью.

## Шаг 5. Подключение к сайту

Откройте файл **config.js** и вставьте URL:

```javascript
window.__LEAD_ENDPOINT__ = 'https://formspree.io/f/ваш_id';
```

Сохраните файл. Готово.

## Проверка

1. Откройте сайт
2. Заполните форму заявки
3. Нажмите «Отправить»
4. Сообщение должно прийти в Telegram

## Лимиты

Formspree бесплатно: до 50 заявок в месяц. Для большего — тариф Pro.
